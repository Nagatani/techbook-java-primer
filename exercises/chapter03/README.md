# 第3章 オブジェクト指向プログラミングの基礎 - 演習課題

## 概要
このディレクトリは第3章「オブジェクト指向プログラミングの基礎」の演習課題です。クラスの設計、オブジェクトの生成、メソッドの実装など、オブジェクト指向プログラミングの基本概念を実践的に学習します。

## 学習目標
- クラスとオブジェクトの関係を理解する
- インスタンス変数とインスタンスメソッドを実装できる
- コンストラクタの役割と実装方法を習得する
- カプセル化の原則を理解し実践する
- getter/setterメソッドを適切に実装できる

## 課題構成

### 基礎課題 (basic/)
1. **BankAccount.java** - 銀行口座クラスの実装
   - 残高管理、入出金機能、残高照会
2. **Rectangle.java** - 長方形クラスの実装
   - 面積・周長計算、サイズ変更機能
3. **Student.java** - 学生情報管理クラス
   - 成績管理、平均点計算機能
4. **Time.java** - 時刻管理クラス
   - 時刻の加算・減算、表示形式の変換

### 発展課題 (advanced/)
- より高度なオブジェクト指向の概念を使用した課題

### チャレンジ課題 (challenge/)
- 複数のクラスを組み合わせた総合的な課題

## 実行方法

### 基本的なコンパイルと実行
```bash
# コンパイル（テストクラスも含む）
javac BankAccount.java BankAccountTest.java

# テストの実行
java BankAccountTest
```

### パッケージを使用する場合
```bash
# srcディレクトリからコンパイル
javac -d . src/*.java

# 実行
java com.example.BankAccountTest
```

## ヒント
- インスタンス変数は`private`で宣言しましょう
- publicメソッドを通じてのみアクセスを許可します
- コンストラクタ内で妥当性検証を行いましょう
- `toString()`メソッドをオーバーライドすると便利です
- 単体テストを書いて動作を確認しましょう
